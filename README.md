# Book Management Fin Project

## Project title
Book Management Fin Project

## Description
This is a Laravel-based book management system that allows authenticated users to create, view, edit, and delete book records. The application includes role-based access control so admins can manage all books and users, while regular users manage only their own books.

The system also includes a dashboard overview showing statistics for total books, total users, and user book activity.

## Implemented features
- User registration and login using Laravel authentication
- Email verification and profile management
- Dashboard overview for authenticated users
- Book management:
  - Create new books
  - View book details
  - Edit existing books
  - Delete books
- Role-based access control for books and users
- Admin access:
  - Manage all books
  - View users and their uploaded books
  - Create, update, and delete users
- Book ownership enforcement using policies
